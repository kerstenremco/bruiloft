Webdev project voor school.
In deze webapplicatie kunnen bruiden:
- een account aanmaken, en cadeau's toevoegen inclusief afbeeldingen. De prioriteit kan worden gewijzigd d.m.v. drag-and-drop
- Gasten uitnodigen. Gasten ontvangen dan een e-mail met een uitnodigingscode waarmee ze in kunnen loggen. Hier kunnen ze aangeven welk cadeau ze hebben gekocht zodat de rest van de gasten deze niet meer koopt. Uiteraard kunnen de bruiden dit zelf niet inzien ;)

Gebruikte techniekenL: HTML, CSS, JS, jQuery, PHP, mySQL
